# 打包压缩

## 1. tar 命令

**tar** 命令用于对文件进行打包压缩或解压，格式：

```shell
tar [选项][文件]
```

| 参数 | 作用                   |
| ---- | ---------------------- |
| -c   | 创建压缩文件           |
| -x   | 解开压缩文件           |
| -t   | 查看压缩包内有哪些文件 |
| -z   | 用 Gzip 压缩或解压     |
| -j   | 用 bzip2 压缩或解压    |
| -v   | 显示压缩或解压的过程   |
| -f   | 目标文件名             |
| -p   | 保留原始的权限与属性   |
| -P   | 使用绝对路径来压缩     |
| -C   | 指定解压到的目录       |

```shell
tar -czvf 压缩包名称.tar.gz 要打包的目录  # 把指定的文件进行打包压缩
tar -xzvf 压缩包名称.tar.gz  # 解压
```
